Site Plan

Site Name

The site name will be weather yourself. I chose this name because I wanted to be like a verb, like when you say improve yourself, but in this case, it means inform yourself about the weather on your location, or in other words, weather yourself.

Site Purpose

The website is created to provide information about the weather and also to share experiences from citizens about the different kinds of weather, tips about how to prepare for a snow storm, a tornado, or any other extreme situation that may occur in some areas. It is created to be easy to use by any people from any age.
